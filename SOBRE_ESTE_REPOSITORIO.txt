DESCRIÇÃO
---------
Servidor MCP (Model Context Protocol) que dá a assistentes de IA (Claude Desktop, Claude
Code, Windsurf, Cursor) acesso COMPLETO à documentação, propriedades e operações dos nós do
n8n — a plataforma de automação de workflows. Com isso, a IA consegue ENTENDER e CONSTRUIR
workflows do n8n por você.

O QUE OFERECE (do README)
-------------------------
- 2.063 nós do n8n (816 core + 1.247 da comunidade, 1.113 verificados)
- Propriedades dos nós (99% de cobertura, com schemas detalhados)
- Operações dos nós (63,6% de cobertura)
- Documentação oficial (87% de cobertura, inclui nós de IA)
- 265 variantes de tools com IA detectadas
- 156 exemplos reais extraídos de templates populares
- Biblioteca de 2.352 templates de workflow (99,96% com metadados de IA)
- Busca de nós da comunidade com filtro por "source"

COMO USAR (resumo)
------------------
Instala-se como servidor MCP no cliente de IA (Claude Desktop/Code, Cursor, Windsurf).
Deploy rápido via npm (n8n-mcp), Docker (ghcr.io/czlonkowski/n8n-mcp) ou Railway (botão).
Opcionalmente conecta a uma instância n8n real (.env) para ler/gerenciar workflows.

UTILIDADE PARA VOCÊ
-------------------
Muito útil se você usa/pretende usar n8n para automações (ex.: pipelines de conteúdo,
alertas, integrações entre suas ferramentas). Permite pedir ao Claude para montar workflows
n8n com conhecimento real dos nós.
